# StrategyScale Use Case Lab

Internal product-testing assets. Not user-facing. Not wired into the app.

Each test card defines a canonical CEO decision, the must-detect constraint, expected good and weak answers, and sample answer sets for pressure-test batches. Used to validate that StrategyScale identifies the underlying constraint rather than the surface decision.

  - Decision mode: Fundraising / Capital
  - Status: Ready for testing
  - Purpose: Tests whether StrategyScale identifies capital readiness rather than capital availability as the real constraint.
